Środowisko: Visual Studio 2022

Technologie: C# , JavaScript, HTML

Platforma chmury obliczeniowej: Microsoft Azure

Developerzy: Wojciech Powroźnik, Patryk Wojciechowski

Link: https://projektweb20220115192315.azurewebsites.net

Zabezpieczenia: Protokół HTTPS

Projekt przedstawia aplikację internetową modelu CRUD - czyli aplikację umożliwiającą utworzenie, aktualizację, modyfikację bądź usunięcie danych czyli niezbędne czynności do zarządzania bazą.

Funkcje zarządzania bazą dostępne są tylko z poziomu administratora - zwykły użytkownik bądź gość ma jedynie możliwość wyświetlania danych z bazy (Tj. Lista Filmów)

Aplikacja umożliwia rejestrację użytkownika.

Administrator ma pełny dostęp do wszystkich funkcjonalności oferowanych przez aplikację (CRUD)

Witryna jest oparta o Platformę Chmury obliczeniowej Microsoft Azure która zapewnia integrację witryny i serwera baz danych.

Microsoft Azure wspomaga też wdrożenie nowych wersji witryny w życie, jest skonfigurowany z systemem kontroli wersji Git, więc pozwala na korzystanie z najnowszych wersji plików.
